# Мой Gemini Telegram Бот

Это умный бот для Telegram, использующий нейросеть **Google Gemini 1.5 Flash**.

## Как запустить:
1. Создайте бота в `@BotFather`.
2. Получите API ключ в [Google AI Studio](https://aistudio.google.com/).
3. Настройте переменные окружения `TELEGRAM_TOKEN` и `GEMINI_API_KEY`.
4. Запустите через `python main.py`.

## Технологии:
* Python 3
* Aiogram 2.x
* Google Generative AI API
